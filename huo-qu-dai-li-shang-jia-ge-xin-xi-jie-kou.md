## 1. 接口描述

调用该接口可以所有系统的价格 /admin/agent/getOriShopPrice

## 2. 输入参数

| 参数名称 | 类型 | 是否必选 | 描述 |
| :--- | :--- | :--- | :--- |
| token | String | 是 | 登录获取的token凭证 |
| username | String | 是 | 用户名（4~32位） |
| password | String | 是 | 添加用户的密码 |
