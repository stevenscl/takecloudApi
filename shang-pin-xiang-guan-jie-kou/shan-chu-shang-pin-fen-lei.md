# 删除商品分类

# 1.接口描述

本接口（admin/category/editCategory）用于删除商品分类。

## 2.输入参数

| 参数名称 | 类型 | 是否必选 | 描述 |
| :--- | :--- | :--- | :--- |
| cid | Integer | 是 | 分类的id |
| deleted | Integer | 是 | 是否删除\(1是\) |
| beid | Integer | 是 | 自己店铺ID |
