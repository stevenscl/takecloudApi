# 商品上架

## 1.接口描述

本接口（admin/goods/startSale）用于小程序的商品的上架。

## 2. 输入参数 {#2-输入参数}

| 参数名称 | 类型 | 是否必传 | 描述 |
| :--- | :--- | :--- | :--- |
| beid | Integer | 是 | 店铺的id |
| goodsId | Integer | 是 | 商品的id |
